# MODEコマンド 全パターン一覧

## 基本セットアップ
NICK testuser
USER testuser 0 * :Test User
JOIN #test

## 1. +i/-i (Invitation Only) パターン

### +i: 招待制設定
MODE #test +i

### -i: 招待制解除
MODE #test -i

## 2. +t/-t (Topic Restriction) パターン

### +t: トピック制限設定
MODE #test +t

### -t: トピック制限解除
MODE #test -t

## 3. +l/-l (User Limit) パターン

### +l: ユーザー制限設定（様々な値）
MODE #test +l 1
MODE #test +l 5
MODE #test +l 10
MODE #test +l 20
MODE #test +l 50
MODE #test +l 100
MODE #test +l 999

### -l: ユーザー制限解除
MODE #test -l

## 4. +k/-k (Channel Key) パターン

### +k: チャンネルキー設定（様々なパスワード）
MODE #test +k password
MODE #test +k secret123
MODE #test +k mykey
MODE #test +k test
MODE #test +k 12345
MODE #test +k abcdefghijklmnopqrstuvwxyz
MODE #test +k "password with spaces"
MODE #test +k password_with_underscores
MODE #test +k password-with-hyphens
MODE #test +k PASSWORD123!@#

### -k: チャンネルキー削除
MODE #test -k

## 5. +o/-o (Operator Privilege) パターン

### 前提: 別ユーザーの参加
# 別のクライアントで実行:
NICK testuser2
USER testuser2 0 * :Test User 2
JOIN #test

NICK testuser3
USER testuser3 0 * :Test User 3
JOIN #test

### +o: オペレーター権限付与
MODE #test +o testuser2
MODE #test +o testuser3

### -o: オペレーター権限剥奪
MODE #test -o testuser2
MODE #test -o testuser3

## 6. 組み合わせパターン（順次実行）

### 複数モードの順次設定
MODE #test +i
MODE #test +t
MODE #test +l 10
MODE #test +k secret
MODE #test +o testuser2

### 複数モードの順次解除
MODE #test -o testuser2
MODE #test -k
MODE #test -l
MODE #test -t
MODE #test -i
